OER Finder Plugin - Example

Explore and discover Open Educational Resources. Compare both routing modes below.

Server-Proxy Mode

1) Via Aggregator Server

Requests are routed through the aggregator backend at http://localhost:3001. The server proxies all adapter calls. Requires the aggregator to be running.

Direct Client Mode

2) Direct Client-Side (No Server)

Adapters run directly in the browser. No aggregator server needed. Each adapter fetches data from its source API directly.
